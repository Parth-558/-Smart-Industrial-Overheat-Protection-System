# 🔥 Smart Industrial Overheat Protection System with Manual Reset

An industrial-grade embedded system designed using Arduino to protect motors from overheating using temperature monitoring, automatic fan control, LCD status display, and manual reset logic. Built for real-world industrial safety scenarios like those used in **Bosch**, **Siemens**, and **Tata** environments.

## 🎯 Project Purpose

In factories and industrial machines, overheating can damage motors and cause safety hazards. This system:

- Detects overheat conditions using **LM35 sensor**
- Automatically shuts down motor
- Activates **cooling fan + buzzer**
- Displays live temperature and system status on LCD
- Requires **manual reset** by operator after cooldown — for real safety

---

## 🔧 Features

✅ Real-time temperature display  
✅ Automatic shutdown at 60°C  
✅ Cooling fan activates automatically  
✅ Yellow LED and buzzer warning  
✅ 30-second cooldown simulation (adjustable)  
✅ Manual reset via push button  
✅ Built-in user interface with 16x2 LCD

## ⚙️ How to Run

1. Open `smart_industrial_overheat_protection_system1.ino` in Arduino IDE or Tinkercad
2. Connect circuit as shown in the image or PDF
3. Simulate LM35 temperature by adjusting voltage
4. At 60°C: Motor stops, red LED + buzzer ON, fan ON
5. Wait 30 seconds → Press reset button to restart

---

## 🧠 System Architecture

[LM35 Sensor] ---> [Arduino UNO] ---> [LCD Display]
|--> [DC Motor (Fan)]
|--> [Red/Green/Yellow LEDs]
|--> [Buzzer]
|--> [Push Button]



🟢 = Normal  
🔴 = Overheat  
🟡 = Cooling  
🔘 = Reset Required
